# Web Engineering 2023-2024 / Lab3 Web API

Please, go to the [GUIDE](docs/GUIDE.md) in order to get the instructions for this assignment.

Some ideas for obtaining a :gift: if you are the first that:

- **Flux**: Support asynchronous requests with [Spring WebFlux](https://docs.spring.io/spring-framework/docs/current/reference/html/web-reactive.html).
- **Be non blocking my friend**: Support asynchronous requests without using **Spring WebFlux**.
- **Secure me basic**: Use HTTP Basic to grant access to request that modify content.
- **Secure me with token**: Use [JWT credentials](https://jwt.io/) to grant access to requests that modify content.
- **Secure me with social**: Use OAuth to grant access to requests that modify content.
